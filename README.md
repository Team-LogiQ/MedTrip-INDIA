# MedTrip INDIA 🏥🇮🇳

### Medical Tourism & Healthcare Discovery Platform

MedTrip INDIA is a **medical tourism and healthcare discovery platform** designed to help patients find the **best hospitals, experienced doctors, and transparent treatment costs across India**.

The platform simplifies the process of planning medical treatment by bringing **hospitals, doctors, treatments, and cost information** together in one place. Patients can explore treatment options, compare hospitals, and make informed healthcare decisions.

This project aims to improve **healthcare accessibility, transparency, and decision-making** for both domestic and international patients seeking medical treatment in India.

---

# 🌐 Live Concept

A centralized platform where users can:

* Search for **medical treatments**
* Explore **top hospitals across Indian cities**
* View **doctor profiles**
* Compare **treatment costs**
* Get **clear healthcare information before planning treatment**

---

# 🎯 Problem Statement

Many patients struggle to:

* Find **reliable hospitals for specific treatments**
* Compare **treatment costs across cities**
* Access **transparent medical information**
* Discover **trusted doctors and healthcare facilities**

Medical tourism is growing rapidly in India, but patients often rely on **multiple disconnected sources of information**.

MedTrip INDIA solves this problem by providing **one unified healthcare discovery platform**.

---

# 💡 Solution

MedTrip INDIA acts as a **healthcare search engine** where patients can:

1. Search for a treatment
2. Select a preferred city
3. Explore hospitals specializing in that treatment
4. View doctor profiles and treatment costs
5. Make an informed decision about their healthcare journey

---

# ⚙️ Installation & Setup

### 1️⃣ Clone the Repository

```
git clone https://github.com/Team-LogiQ/MedTrip-INDIA.git
```

### 2️⃣ Navigate to Project Folder

```
cd MedTrip-INDIA
```

### 3️⃣ Setup Environment Variables

Create a `.env` file based on `.env.example`.

Example:

```
API_KEY=your_api_key
AUTH_SECRET=your_secret
```

---

### 4️⃣ Run the Project

Simply open the project folder and run the main HTML file in a browser.

Example:

```
index.html
```
